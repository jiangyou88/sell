## MVVM框架
- View:视图 DOM
- ViewModel:通讯 观察者
- Model:数据 JavaScript对象
- 针对具有复杂交互逻辑的前端应用
- 提供基础的架构抽象
- 通过Ajax数据持久化，保证前端用户体验
